Alpha / active development

Docs that match the product we are actually building.

Vedetta is a self-hosted, DNS-first security monitoring platform. Today it is strongest at device discovery, passive DNS visibility, and local threat scoring for homes and small businesses.

Pi-hole and AdGuard Home are optional integrations. Router and firewall aggregation is part of the next expansion. The community threat network is future-facing, not the main promise of the current product.

What Vedetta Is Today

The current product shape in plain language.

Core runs in Docker. A native sensor provides the strongest current visibility. The local deployment should stand on its own, with optional integrations layered on top.

Current wedge

DNS-first visibility and detection

Vedetta already captures and scores DNS activity locally, then enriches that data with local threat intelligence.

Current wedge

Device discovery and inventory

The native sensor finds devices, reports them back to Core, and gives the dashboard a useful local inventory.

Optional inputs

Pi-hole and AdGuard Home integrations

Both are optional sources if you already run them. They add value, but they are not required to understand Vedetta.
