Why Vedetta

See everything. Understand it simply.

Device Discovery

Automatically find every device on your network — phones, cameras, smart home gear, servers. Know what's connected and when something new shows up.

Community Threat Intel

Opt in to contribute anonymized telemetry to a shared threat network. Every Vedetta instance makes every other instance smarter. Strength in numbers.

Runs Anywhere

Deploy on a Raspberry Pi, a NUC, or a Mac Studio. Docker-based Core with native sensors. Five minutes from git clone to full network visibility.

DNS Monitoring

Passive DNS analysis via Pi-hole integration. See where your devices are calling home, catch suspicious domains, and track query patterns over time.

Core + Sensor

Core manages the dashboard, database, and intelligence. Sensors are lightweight binaries on each network segment with full L2 fingerprinting.

Open Source

AGPLv3 licensed. Audit the code, fork it, extend it. No vendor lock-in, no subscriptions. Your network data stays on your hardware.

Get Started

Five minutes to full visibility.

01 — CORE

Start the watchtower

Clone the repo and spin up Core with Docker Compose. Dashboard and API come up automatically.

git clone https://github.com/MahdiHedhli/vedetta.git
cd vedetta
docker compose up -d
02 — SENSOR

Deploy a sensor

Run the one-liner on any machine on your LAN. It installs dependencies, builds from source, and registers as a persistent service.

curl -fsSL https://raw.githubusercontent.com/
MahdiHedhli/vedetta/main/sensor/
deploy/install.sh | sudo bash -s --
--core http://<CORE_IP>:8080
